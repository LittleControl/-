# 流行音乐管理系统

## 实现的功能

1. 数据库存数储功能
2. 数据排序
3. 首页的登录检测
4. 基本的CRUD

## 相关技术

- Node(后台编写语言)
- Mongoose(MongoDB)
- Bootstrap(前端相关框架)
- Express(Node插件)
- 排序算法(快速排序)
